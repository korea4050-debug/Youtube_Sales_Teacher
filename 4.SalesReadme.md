# 🤖 ChatGPT API + Streamlit을 활용한 나만의 AI 비서 만들기

이 프로젝트는 OpenAI의 ChatGPT API와 파이썬 웹 프레임워크인 Streamlit을 사용하여 대화 내역을 기억하고 실시간으로 응답하는 챗봇 웹 애플리케이션을 만드는 프로젝트입니다.

## 📌 준비물
1. **Python 설치**: Python 3.8 버전 이상이 필요합니다.
2. **OpenAI API Key**: [OpenAI Platform](https://platform.openai.com/)에서 발급받아야 합니다. (유료 크레딧 필요)
3. **코드 에디터**: VS Code를 권장합니다.

---

가장 먼저 프로젝트 파일을 담을 폴더를 만들고 터미널(또는 CMD)에서 해당 경로로 이동합니다.

```bash
# 가상환경 활성화
# Windows:
.\venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate
```

* `python-dotenv`: API 키를 환경변수로 안전하게 관리하기 위한 라이브러리

## 🔑 3. 환경 변수 설정 (.env)
보안을 위해 API 키는 코드에 직접 적지 않고 별도의 파일에 저장합니다. 폴더 안에 `.env` 파일을 만들고 아래 내용을 입력하세요.

**[.env]**
```text
OPENAI_API_KEY=your_api_key_here
```
*(참고: `your_api_key_here` 부분을 실제 발급받은 `sk-...` 형태의 키로 교체하세요.)*

## 💻 4. 메인 코드 작성 (main.py)
이제 메인 프로그램 코드를 작성합니다. `main.py` 파일을 만들고 아래 코드를 복사해 넣으세요.

**[main.py]**
```python
import streamlit as st
from openai import OpenAI
import os
from dotenv import load_dotenv

# .env 파일에 저장된 환경 변수 로드
load_dotenv()

# OpenAI 클라이언트 초기화
client = OpenAI(api_key=os.getenv("OPENAI_API_KEY"))

# 1. 페이지 설정 및 제목
st.set_page_config(page_title="나만의 AI 비서", page_icon="🤖")
st.title("🤖 나만의 AI 비서")
st.caption("OpenAI GPT-4o-mini를 활용한 챗봇입니다.")

# 2. 대화 내역 초기화 (세션 상태 관리)
# Streamlit은 새로고침될 때 코드가 처음부터 다시 실행되므로, 대화 내역을 유지하기 위해 session_state를 사용합니다.
if "messages" not in st.session_state:
    st.session_state.messages = [
        {"role": "system", "content": "당신은 유능하고 친절한 AI 비서입니다."}
    ]

# 3. 기존 대화 내역 화면에 출력
for message in st.session_state.messages:
    if message["role"] != "system":
        with st.chat_message(message["role"]):
            st.markdown(message["content"])

# 4. 사용자 입력 받기
if prompt := st.chat_input("메시지를 입력하세요"):
    # 사용자 메시지를 세션에 저장 및 화면 표시
    st.session_state.messages.append({"role": "user", "content": prompt})
    with st.chat_message("user"):
        st.markdown(prompt)

    # 5. AI 응답 생성 및 표시
    with st.chat_message("assistant"):
        message_placeholder = st.empty() # 답변이 채워질 빈 공간 생성
        full_response = ""
        
        # OpenAI API 호출 (스트리밍 방식)
        for response in client.chat.completions.create(
            model="gpt-4o-mini", # 비용 효율적인 최신 모델 사용
            messages=[
                {"role": m["role"], "content": m["content"]}
                for m in st.session_state.messages
            ],
            stream=True,
        ):
            # 스트리밍 데이터에서 텍스트 추출
            content = response.choices[0].delta.content
            if content:
                full_response += content
                message_placeholder.markdown(full_response + "▌") # 타이핑 효과
        
        message_placeholder.markdown(full_response) # 최종 답변 확정

    # AI 답변을 세션 내역에 추가
    st.session_state.messages.append({"role": "assistant", "content": full_response})
```

---

## 🚀 5. 프로그램 실행
터미널에서 다음 명령어를 입력하여 웹 애플리케이션을 실행합니다.

```bash
streamlit run main.py
```

실행하면 브라우저 창이 자동으로 열리며 `http://localhost:8501` 주소에서 당신의 AI 비서와 대화를 시작할 수 있습니다.

---

## 💡 주요 포인트 설명
1.  **st.session_state**: 대화는 여러 번의 질의응답으로 이루어집니다. Streamlit은 사용자가 입력을 보낼 때마다 코드를 재실행하는데, 이때 대화 기록이 지워지지 않도록 메모리 역할을 하는 세션 저장소를 사용합니다.
2.  **System Role**: 코드의 `system` 메시지 부분을 "당신은 무뚝뚝한 요리사입니다" 등으로 바꾸면 AI의 성격이 즉시 변합니다.
3.  **Streaming**: `stream=True` 옵션을 통해 ChatGPT 웹사이트처럼 글자가 하나씩 실시간으로 써지는 효과를 구현했습니다.
4.  **보안**: `.env` 파일을 통해 API 키 유출을 방지합니다. (GitHub에 올릴 때는 `.gitignore`에 `.env`를 추가하세요.)
