# 📺 AI YouTube Summarizer 구축 가이드 (PocketFlow 기반)

이 프로젝트는 긴 유튜브 영상의 자막을 추출하고, 이를 핵심 주제별로 분류(Identify Topics)한 뒤, 각 섹션을 요약하여 시각적인 HTML 보고서로 만드는 도구입니다.

## 1. 사전 준비 사항
*   **Python 3.9+** 설치
*   **OpenAI API Key**: [platform.openai.com](https://platform.openai.com/)에서 발급
*   **Cursor AI (권장)**: 에이전트 기능을 활용해 코드를 자동 생성하기 위해 필요
*   **필수 라이브러리**:
    ```bash
    pip install pocket-flow openai youtube-transcript-api beautifulsoup4 requests python-dotenv
    ```

## 3. 단계별 구현

### 단계 1: 환경 변수 설정 (`.env`)
```env
OPENAI_API_KEY=your_openai_api_key_here
```

### 단계 2: 설계도 작성 (`design.md`)
이 파일은 Cursor AI의 Composer(에이전트)에게 프로젝트의 전체 흐름을 설명하는 용도입니다.

```markdown
# 프로젝트 개요: YouTube Summarizer Flow
PocketFlow 프레임워크를 사용하여 유튜브 요약 에이전트를 구축한다.

## 흐름(Flow) 단계:
1. **ExtractInfoNode**: 유튜브 URL을 입력받아 제목, 썸네일, 자막(Transcript)을 추출한다.
2. **IdentifyTopicsNode**: 전체 자막을 분석하여 5~10개의 주요 주제(Topics) 리스트를 생성한다.
3. **SummarizeBatchNode (Map)**: 각 주제별로 관련 자막 내용을 할당하고 요약 및 핵심 Q&A를 생성한다.
4. **GenerateHTMLNode (Reduce)**: 요약된 모든 정보를 합쳐 시각적으로 보기 좋은 `output.html` 파일을 생성한다.
```

### 단계 3: 유틸리티 함수 작성 (`utils.py`)
유튜브 자막과 메타데이터를 가져오는 함수입니다.

```python
import re
import requests
from bs4 import BeautifulSoup
from youtube_transcript_api import YouTubeTranscriptApi

def extract_video_id(url):
    return re.search(r"(?:v=|\/)([0-9A-Za-z_-]{11}).*", url).group(1)

def get_video_metadata(url):
    response = requests.get(url)
    soup = BeautifulSoup(response.text, 'html.parser')
    title = soup.find("title").text.replace(" - YouTube", "")
    video_id = extract_video_id(url)
    return {
        "title": title,
        "thumbnail": f"https://img.youtube.com/vi/{video_id}/maxresdefault.jpg"
    }

def get_transcript(video_id):
    try:
        transcript_list = YouTubeTranscriptApi.get_transcript(video_id, languages=['ko', 'en'])
        return " ".join([t['text'] for t in transcript_list])
    except Exception as e:
        return f"자막을 가져올 수 없습니다: {str(e)}"
```

### 단계 4: PocketFlow 로직 구현 (`flow.py`)
이 부분은 Cursor AI의 **Composer (Ctrl+I)**를 열고 다음과 같이 명령하여 생성하는 것이 가장 빠릅니다.

> **Cursor에게 주는 프롬프트:**
> "PocketFlow 프레임워크를 사용해서 `design.md`에 정의된 노드들을 `flow.py`에 구현해줘. `utils.py`의 함수들을 활용하고 OpenAI GPT-4o 모델을 사용해줘."

**(참고용 핵심 코드 구조)**
```python
from pocketflow import Node, BatchNode, Flow
from utils import get_video_metadata, get_transcript, extract_video_id
import openai

# 주제별 요약을 수행하는 노드 (Map 단계)
class SummarizeBatchNode(BatchNode):
    def exec(self, item):
        # 개별 주제에 대한 GPT 요약 로직
        pass

# 전체 흐름 정의
def create_flow():
    flow = Flow()
    flow.add_node("extract", ExtractInfoNode())
    flow.add_node("identify", IdentifyTopicsNode())
    flow.add_node("summarize", SummarizeBatchNode(), map_to="topics")
    flow.add_node("html", GenerateHTMLNode())
    
    flow.set_start_node("extract")
    flow.connect("extract", "identify")
    flow.connect("identify", "summarize")
    flow.connect("summarize", "html")
    return flow
```

### 단계 5: 실행 파일 작성 (`main.py`)
```python
from flow import create_flow
from dotenv import load_dotenv

load_dotenv()

if __name__ == "__main__":
    url = input("요약할 유튜브 URL을 입력하세요: ")
    
    shared_store = {"url": url}
    flow = create_flow()
    flow.run(shared_store)
    
    print("✅ 요약 완료! output.html 파일을 확인하세요.")
```

## 4. Cursor AI를 활용한 실전 팁 (Vibe Coding)

1.  **PocketFlow 원본 코드 참조**: PocketFlow는 매우 작습니다. Cursor에게 `pocket-flow` 라이브러리의 소스 코드나 문서를 `@` 기호로 참조해주면 훨씬 정확하게 코드를 짭니다.
2.  **Map-Reduce 패턴**: 영상에서 강조하는 핵심입니다. 긴 자막을 한 번에 요약하려 하지 말고, 반드시 **주제 추출(Identify Topics) -> 주제별 요약(Batch Summarize)** 과정을 거치도록 설계하세요.
3.  **프롬프트 수정**: `IdentifyTopicsNode`에서 "이 영상을 7개의 짧은 챕터로 나눠줘"라고 명시하면 결과물의 가독성이 좋아집니다.
