# 🤖 ChatGPT API + Streamlit을 활용한 AI 비서 만들기

이 프로젝트는 OpenAI의 API와 Streamlit 프레임워크를 사용하여 웹 브라우저에서 동작하는 나만의 AI 챗봇을 만드는 과정을 다룹니다.

## 📌 준비 사항
1. **Python 설치**: Python 3.8 버전 이상이 필요합니다.
2. **OpenAI API Key**: [OpenAI Platform](https://platform.openai.com/)에서 API 키를 발급받아야 합니다.
3. **IDE**: VS Code 또는 PyCharm 사용을 권장합니다.

---

## 📂 1. 프로젝트 구조 설정
먼저 프로젝트를 진행할 폴더를 생성하고 필요한 파일을 만듭니다.

```bash
mkdir my-ai-assistant
```

```text

```

## 💻 4. 메인 코드 작성 (main.py)
Streamlit의 화면 구성과 OpenAI API 호출 로직을 작성합니다. 이 코드는 대화 내역을 기억하는 기능을 포함합니다.

**[main.py]**
```python
import streamlit as st
from openai import OpenAI
import os
from dotenv import load_dotenv

# .env 파일에서 환경변수 로드
load_dotenv()

# OpenAI 클라이언트 초기화
client = OpenAI(api_key=os.getenv("OPENAI_API_KEY"))

# 페이지 설정
st.set_page_config(page_title="나만의 AI 비서", page_icon="🤖")
st.title("🤖 나만의 AI 비서")
st.caption("무엇이든 물어보세요!")

# 세션 상태에 메시지 저장소 초기화 (대화 내역 유지)
if "messages" not in st.session_state:
    st.session_state.messages = [
        {"role": "system", "content": "당신은 친절하고 유능한 비서입니다."}
    ]

# 저장된 메시지 화면에 표시 (System 메시지 제외)
for message in st.session_state.messages:
    if message["role"] != "system":
        with st.chat_message(message["role"]):
            st.markdown(message["content"])

# 사용자 입력 받기
if prompt := st.chat_input("메시지를 입력하세요..."):
    # 1. 사용자 메시지 추가 및 화면 표시
    st.session_state.messages.append({"role": "user", "content": prompt})
    with st.chat_message("user"):
        st.markdown(prompt)

    # 2. AI 응답 생성 및 화면 표시
    with st.chat_message("assistant"):
        message_placeholder = st.empty()
        full_response = ""
        
        # OpenAI API 호출 (스트리밍 방식)
        response = client.chat.completions.create(
            model="gpt-3.5-turbo", # 또는 "gpt-4o"
            messages=[
                {"role": m["role"], "content": m["content"]}
                for m in st.session_state.messages
            ],
            stream=True,
        )
        
        for chunk in response:
            if chunk.choices[0].delta.content is not None:
                full_response += chunk.choices[0].delta.content
                message_placeholder.markdown(full_response + "▌")
        
        message_placeholder.markdown(full_response)
    
    # 3. AI 응답을 세션 상태에 저장
    st.session_state.messages.append({"role": "assistant", "content": full_response})
```

## 🚀 5. 프로그램 실행
터미널에서 다음 명령어를 입력하여 스트림릿 서버를 실행합니다.

```bash
streamlit run main.py
```

명령어를 실행하면 로컬 웹 주소(보통 `http://localhost:8501`)가 뜨며 브라우저에서 AI 비서와 대화할 수 있습니다.

---

## 💡 주요 포인트 설명

1.  **st.session_state**: Streamlit은 버튼을 누르거나 입력할 때마다 코드가 처음부터 다시 실행됩니다. 변수 값을 유지하기 위해 `session_state`를 사용하여 대화 내역(`messages`)을 저장합니다.
2.  **System Role**: `messages` 리스트의 첫 번째 항목에 "당신은 친절한 요리사입니다"와 같이 역할을 부여하면 AI의 성격이 바뀝니다.
3.  **Stream**: `stream=True` 옵션을 주면 답변이 한 번에 나오지 않고 ChatGPT처럼 실시간으로 써지는 효과를 줍니다.
4.  **python-dotenv**: API 키 노출 방지를 위해 `.env` 파일을 읽어오는 라이브러리입니다.
