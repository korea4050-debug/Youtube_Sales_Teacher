# 🚀 AI YouTube Summarizer 구축 가이드 (with PocketFlow)

이 프로젝트는 긴 유튜브 영상의 자막을 추출하여 핵심 주제별로 요약하고, 이를 시각적인 HTML 페이지로 생성하는 도구입니다. **PocketFlow**라는 100줄 내외의 경량 프레임워크를 사용하여 복잡한 에이전트 구조를 간단하게 구현합니다.

## 1. 사전 준비 (Prerequisites)
*   **Cursor AI**: 에이전트 기능을 활용하기 위해 설치 권장.
*   **OpenAI API Key**: 자막 요약 및 주제 추출을 위해 필요.
*   **Python 3.9+**: 시스템에 설치되어 있어야 함.

## 3. 핵심 아키텍처 설계 (Design Doc)
PocketFlow의 핵심은 **"먼저 설계하고, 에이전트에게 구현을 맡기는 것"**입니다. 프로젝트 루트에 `design.md`를 작성하여 에이전트(Cursor)가 흐름을 이해하게 합니다.

```markdown
# Flow Design: YouTube Summarizer
1. **InputNode**: 유튜브 URL을 입력받음.
2. **ExtractInfoNode**: 유튜브 제목, 썸네일, 전체 자막을 추출.
3. **IdentifyTopicsNode**: 자막을 분석하여 주요 주제(Topics) 리스트 생성.
4. **SummarizeBatchNode (Map)**: 각 주제별로 자막 섹션을 할당하여 요약 및 Q&A 생성.
5. **GenerateHTMLNode (Reduce)**: 요약된 내용을 바탕으로 시각적인 HTML 결과물 생성.
```

## 4. 단계별 구현 (Implementation)

### Step 1: 유틸리티 함수 작성 (`utils.py`)
유튜브 정보를 가져오는 기본 함수들을 작성합니다.

```python
import re
import requests
from bs4 import BeautifulSoup
from youtube_transcript_api import YouTubeTranscriptApi

def extract_video_id(url):
    return re.search(r"(?:v=|\/)([0-9A-Za-z_-]{11}).*", url).group(1)

def get_video_metadata(url):
    response = requests.get(url)
    soup = BeautifulSoup(response.text, 'html.parser')
    return {
        "title": soup.find("title").text,
        "thumbnail": f"https://img.youtube.com/vi/{extract_video_id(url)}/maxresdefault.jpg"
    }

def get_transcript(video_id):
    transcript_list = YouTubeTranscriptApi.get_transcript(video_id, languages=['ko', 'en'])
    return " ".join([t['text'] for t in transcript_list])
```

### Step 2: PocketFlow 노드 정의 (`flow.py`)
Cursor AI의 `Composer` (Ctrl+I)를 열고 다음과 같이 요청하세요:
> "PocketFlow 프레임워크를 사용하여 design.md에 정의된 대로 flow.py를 구현해줘. 각 노드는 OpenAI GPT-4o를 사용해야 해."

**주요 코드 구조 (Flow 예시):**
```python
from pocketflow import Node, BatchNode, Flow
from utils import get_video_metadata, get_transcript, extract_video_id

class ExtractInfoNode(Node):
    def prep(self, shared):
        return shared['url']
    def exec(self, url):
        vid = extract_video_id(url)
        meta = get_video_metadata(url)
        transcript = get_transcript(vid)
        return {"meta": meta, "transcript": transcript}
    def post(self, shared, res):
        shared.update(res)

# ... IdentifyTopicsNode, SummarizeBatchNode 등 추가 구현 ...

def create_flow():
    flow = Flow()
    flow.add_node("extract", ExtractInfoNode())
    # 노드 간 연결 정의
    return flow
```

## 5. 프롬프트 엔지니어링 (Map-Reduce 패턴)
긴 영상을 한 번에 요약하면 정보가 누락됩니다. 영상에서 강조하는 **Map-Reduce** 기법을 적용하세요.

1.  **Map (Identify Topics)**: "이 자막을 보고 5~10개의 독립적인 주제 리스트로 나눠줘."
2.  **Batch Process**: 각 주제별로 연관된 자막만 모아서 개별 요약 생성.
3.  **Reduce (HTML)**: 모든 요약을 하나로 합쳐서 HTML 파일로 저장.

## 6. 실행 및 결과 확인 (`main.py`)
```python
from flow import create_flow

if __name__ == "__main__":
    yt_url = input("유튜브 URL을 입력하세요: ")
    flow = create_flow()
    
    # 초기 데이터 세팅 및 실행
    shared_store = {"url": yt_url}
    flow.run(shared_store)
    
    print("요약이 완료되었습니다! output.html을 확인하세요.")
```
